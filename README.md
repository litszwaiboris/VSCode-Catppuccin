<h1>Installation</h1>

Put the `workbench.html` file in `Your VSCode installation\resources\app\out\vs\code\electron-sandbox\workbench\`
Put the `code-icon-catppuccin.svg` file in `Your VSCode installation\resources\app\out\media\`
